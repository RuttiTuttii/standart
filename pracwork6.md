5. Рекомендации по оформлению в LibreOffice Writer

1. Вставка таблиц:
   · Скопируйте текст с разделителем-табуляцией (или Markdown).
   · Выделите скопированный текст.
   · Выберите меню «Таблица» → «Преобразовать» → «Текст в таблицу».
   · Укажите разделитель (табуляция или другой символ).
   · Нажмите OK.
2. Схемы:
   · Скопируйте ASCII-блок.
   · Установите для этого абзаца шрифт Courier New, размер 10–11 пт.
   · Выровняйте по центру.
   · При желании можно вставить как рисунок, сняв скриншот, но текстовый вариант удобнее для редактирования.
3. Заголовки и нумерация:
   · Заголовки разделов (ВВЕДЕНИЕ, 1, 2 и т.д.) наберите полужирным шрифтом размером 16, по центру.
   · Заголовки подразделов – полужирным шрифтом размером 14, с абзацного отступа.
4. Страницы:
   · Нумерацию страниц вставьте через «Вставка» → «Нижний колонтитул» → «Обычный» и выберите вставку номера страницы по центру.

Если вам нужна помощь с форматированием конкретного элемента (например, как сделать так, чтобы таблица не разрывалась, или как добавить рамку), пишите — подскажу.
